## **👋Introduction**

> 오픈소스 컨트리뷰터들을 위한 커뮤니티 op-in  을 소게합니다.
op-in은 오픈소스 기여자의 기여자를 위해 기여자에 의한 커뮤니티 서비스입니다
op-in 은 git-hub 의 래포지토리를 기반으로 POST, QnA 를 작성하고 소통하면서
오픈소스 기여에 조금더 활성화 해보고 싶어 만들어진 서비스입니다.
